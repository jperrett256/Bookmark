# Bookmark

An extension for saving scroll positions.

## Setup

1. Clone the repository
2. Package everything (except this README) into a zip file (with no compression)

The packaged extension can now be signed and used.

## Notes

- This was written for Firefox
- Uses [Monsterrat](https://fonts.google.com/specimen/Montserrat) font
- This was tested with [jQuery](https://jquery.com/download/) version 3.4.1
